AI-Powered Customer Support Chat Application

This project is a full-stack web application where users can interact with an AI-powered assistant.
Admins can upload FAQs, PDF files, and text documents, and the AI uses those documents along with its general knowledge to provide accurate and context-aware replies.

The application includes a React + MobX frontend, Node.js + Express backend, MongoDB database, and LLM API integration (OpenAI, Gemini, Claude, DeepSeek).
